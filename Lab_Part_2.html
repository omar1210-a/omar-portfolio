import geopandas as geopandas
import matplotlib.pyplot as plt

states = geopandas.read_file(r"Data/110m_cultural/ne_110m_admin_1_states_provinces.shp")

fires_8599 = geopandas.read_file(r"Data/wildfires_1985-1999/wildfires_1985-1999.shp")
fires_0014 = geopandas.read_file(r"Data/wildfires_2000-2014/wildfires_2000-2014.shp")

fires_8599 = fires_8599.to_crs(states.crs)
fires_0014 = fires_0014.to_crs(states.crs)

j_8599 = geopandas.sjoin(states[['geometry']], fires_8599, how='left', predicate='contains')
counts_8599 = j_8599.groupby(j_8599.index).size()

states_8599 = states.copy()
states_8599['number_of_fires'] = counts_8599
states_8599['number_of_fires'] = states_8599['number_of_fires'].fillna(0).astype(int)

j_0014 = geopandas.sjoin(states[['geometry']], fires_0014, how='left', predicate='contains')
counts_0014 = j_0014.groupby(j_0014.index).size()

states_0014 = states.copy()
states_0014['number_of_fires'] = counts_0014
states_0014['number_of_fires'] = states_0014['number_of_fires'].fillna(0).astype(int)

states_8599[['name', 'number_of_fires']]
states_0014[['name', 'number_of_fires']]

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(15, 5))

To complete this analysis, I first examined the two wildfire shapefiles representing the periods 19851999 and 20002014, along with the U.S. states boundary shapefile. Using what I learned from the coursework where we followed along, I read each dataset with geopandas.read_file() and ensured they shared the same coordinate reference system with to_crs(). Then, I used a spatial join (geopandas.sjoin) to connect each wildfire point to the state polygon it falls within. After the join, I applied groupby() to count how many wildfires occurred in each state and attached these totals back to the states layer as a new column called number_of_fires. Finally, I plotted the results as two separate choropleth maps using the plot() function with the OrRd color map and a legend to visualize the distribution of wildfires for each time period.
